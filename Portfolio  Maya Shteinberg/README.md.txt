# מאיה שטיינברג - תיק עבודות אישי

## תיאור
קובץ זה מכיל את תיק העבודות אישי, המציג את פרויקטי הפיתוח והכישורים בתחום פיתוח אתרים. 
תיק העבודות נבנה באמצעות HTML, CSS ו-JavaScript, וכולל מגוון של פרויקטים ודוגמאות מעניינות.


## התחלה
כדי לצפות בתיק העבודות, פשוט פתח את קובץ `index.html` בדפדפן האינטרנט שלך.


## טכנולוגיות בשימוש
תיק העבודות נבנה באמצעות הטכנולוגיות הבאות:
- HTML
- CSS
- JavaScript
- Bootstrap (לעיצוב)
